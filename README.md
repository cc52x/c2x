# My Responsive Website

A modern, fully responsive website that works seamlessly on mobile phones, tablets, and desktop computers.

## 📋 Features

- **Responsive Design** - Adapts to any screen size
- **Mobile First** - Optimized for mobile devices first, then scaled up
- **Interactive Elements** - Hamburger menu, smooth scrolling, and buttons
- **Clean Code** - Easy to understand and modify
- **Fast Loading** - Minimal dependencies, pure HTML/CSS/JavaScript

## 📁 Project Structure

```
c2x/
├── index.html          # Main HTML file with page structure
├── styles.css          # Responsive CSS styling
├── script.js           # JavaScript for interactivity
└── README.md           # This file
```

## 🚀 Getting Started

### Option 1: Open Locally
1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start editing the files!

### Option 2: Use a Local Server (Recommended)
```bash
# If you have Python 3
python -m http.server 8000

# If you have Python 2
python -m SimpleHTTPServer 8000

# Then open: http://localhost:8000
```

## 📱 Responsive Breakpoints

The website uses these breakpoints for responsive design:

- **Desktop**: 1200px+ (full layout)
- **Tablet**: 769px - 1199px (adjusted layout)
- **Mobile**: Below 768px (mobile-optimized layout)
- **Small Mobile**: Below 480px (extra small adjustments)

## 🎨 Customization

### Change Colors
Edit `styles.css` and look for color values like:
- `#2c3e50` - Dark blue/gray
- `#667eea` - Purple/blue gradient
- `#3498db` - Light blue

### Change Text
Edit `index.html` to update:
- Navigation links
- Section headings
- Content text

### Add More Sections
Copy a section (like the "About" section) and modify the class names and content.

## 💡 What Each File Does

### index.html
- Contains the HTML structure
- Includes Navigation, Hero, About, Services, Contact, and Footer sections
- Links to CSS and JavaScript files

### styles.css
- Contains all visual styling
- Responsive media queries for different screen sizes
- Hover effects and animations

### script.js
- Makes the hamburger menu work
- Enables smooth scrolling
- Handles button clicks
- Logs information to console
